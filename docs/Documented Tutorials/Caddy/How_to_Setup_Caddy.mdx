## Setting up Caddy

<Tabs>
  <TabItem value="Setting up Caddy" label="Basic Setup" default>

**Caddy Setup:**

**Caddyfile Download**

[Wordpress Caddyfile](https://github.com/DevilsDesigns/Youtube-Docs/blob/main/Caddy-Examples/Wordpress/Caddyfile)

**Running Caddy inside the Caddy.exe Directory**

1. 

```css
cd C:\Caddy\Directory
```
    ie. cd C:\Tools\Caddy

2. Running Caddy Manually

```css
./caddy run --config Caddyfile
```

:::caution

Do not Forget to **Accept both permissions if a windows Pops up** upon first run.

:::

**Setting Up Cloudflare Tunnels:**

**Downloads**

[CloudFlare DDNS](https://github.com/wyattjoh/cloudflare-ddns/releases)

**Setting up A Name Record:**
1. Download Cloudflare DDNS from github and extract it to a directory of your choice ie. `C:\Tools\CloudflareDDNS`

2. Open https://dash.cloudflare.com/ and select your domain

3. Click DNS on the side bar 

![image](https://user-images.githubusercontent.com/35183970/215296063-666a2625-e377-4b03-878f-4856695da230.png)

4. Click add a Record

![image](https://user-images.githubusercontent.com/35183970/215296074-272ba4a3-759c-4c66-9ec4-6f367b1d3f3d.png)

5. Go to https://ipchicken.com/ and copy your external ip into the A NAME ie.

![image](https://user-images.githubusercontent.com/35183970/215296110-3cc4af57-e544-4726-9b6a-d2710eb51296.png)

6. Click Save

**Getting an API Key:**


1. Go to Overview in the right Hand Menu

![image](https://user-images.githubusercontent.com/35183970/215296144-c99839f2-bf53-4397-b9e0-a47a0d202d18.png)

2. Scroll Down till you see get API Token and click it

![image](https://user-images.githubusercontent.com/35183970/215296159-911009a0-de4b-4f0f-8f68-d758af348a71.png)

3. Click Create a Token

![image](https://user-images.githubusercontent.com/35183970/215296175-26bfe947-0d46-442a-9861-13f24f63611b.png)

4. Scroll down and select Custom Token

![image](https://user-images.githubusercontent.com/35183970/215296224-f911035a-0f96-4426-8673-43cdd7bf0e53.png)
 
5. Name the Token, Give it Zone, Zone, Read and then click add more and add Zone, DNS, Edit Like so

![image](https://user-images.githubusercontent.com/35183970/215296326-eae964e9-4f20-4132-98a6-07e554d2259e.png)

6. Then Click Create Token

7. Create Copy your token somewhere safe

![image](https://user-images.githubusercontent.com/35183970/215296333-b790ded6-84fd-46c6-93f0-b4911fb58284.png)

**Running Cloudflare DDNS:**

1. type in powershell 

```css
cd C:\Tools\CloudflareDDNS
```

2. Run this command after you edit the code to match yours

```css
./cloudflare-ddns --token your-cloudflare-api-key --domain home.yourdomain.com
```

ie. 
`./cloudflare-ddns --token apkGKJHDkansjhcbsdjhc --domain home.demonwarriortech.com`

3. it should get a log like this

```css
time="2023-01-28T17:54:39-06:00" level=info msg="updated record" content=123.123.123.123 name=home.ittechnut.com
PS C:\tools\cloudflareddns>
```

**Autostartup CloudflareDDNS:**

1. Search Run in Windows

![image](https://user-images.githubusercontent.com/35183970/215296525-850af871-bae2-427d-b641-f4e73f564c89.png)

2. Type `shell:startup` and enter it will open a folder with your startup apps. These are all the apps that start when windows Starts

3. Right Click anywhere in empty space in your folder and click new ---> Text Documents and name it CloudFlareDDNS.bat **(Make sure You do Not save it as a .txt)**

![image](https://user-images.githubusercontent.com/35183970/215296594-5d2ac03c-570f-49a3-9993-3ba829d2744f.png)

4. Now open the text document and paste

```css
C:\Tools\CloudFlareDDNS\cloudflare-ddns --token your-cloudflare-api-token--domain home.yourdomain.com
pause
```

ie.

```css
C:\Tools\CloudFlareDDNS\cloudflare-ddns --token apkGKJHDkansjhcbsdjhc --domain home.demonwarriortech.com
pause
```

5. Click Save or CTRL+S

6. Run it and it should now be autorunning your background
